# Portafolio personal — André Vicente

Portafolio web creado para presentar mi perfil, habilidades y proyectos como desarrollador backend junior y estudiante de Ingeniería de Sistemas.

## Contenido

- Presentación profesional
- Perfil y stack actual
- Proyectos personales
- Enlaces a los repositorios de cada proyecto

## Tecnologías

- HTML5
- CSS3
- Git y GitHub

## Ejecución local

1. Clona el repositorio:

   ```bash
   git clone https://github.com/lAndreV10/lAndreV10.io.git
   ```

2. Entra en la carpeta del proyecto:

   ```bash
   cd lAndreV10.io
   ```

3. Abre `index.html` en tu navegador.
